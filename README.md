# Netflix India Clone 🎬

A static front-end clone of the Netflix India landing page, built with pure HTML and CSS.

---

## 📸 Overview

This project replicates the Netflix India homepage UI, including the hero section, feature highlights, FAQ section, and footer — with no frameworks or JavaScript.

---

## 🗂️ Project Structure

```
netflix-clone/
├── index.html
├── STYLE.CSS
└── assets/
    └── images/
        └── logo.svg
```

---

## ✨ Features

- **Navbar** — Netflix logo, language selector button, and Sign In button
- **Hero Section** — Headline, subtext, and email input with a "Get Started" CTA
- **Feature Sections** — Four alternating image/text sections covering:
  - Enjoy on your TV
  - Download shows to watch offline
  - Watch everywhere (multi-device)
  - Create profiles for kids
- **FAQ Section** — Four expandable FAQ items (accordion-style UI)
- **Footer** — Help links grid and contact number

---

## 🚀 Getting Started

1. **Clone or download** this repository.
2. Open `index.html` in any modern browser — no build step or server required.

```bash
# Optional: serve locally with Python
python -m http.server 8000
```

Then visit `http://localhost:8000` in your browser.
